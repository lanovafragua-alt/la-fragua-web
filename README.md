# La Fragua Web

Primera versión de la web principal de La Fragua, creada con Next.js, Tailwind CSS y App Router para despliegue en Vercel.

## Rutas

- `/`
- `/laboratorio`
- `/ascuas`
- `/focus-mode`
- `/productos`
- `/contacto`

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Producción

```bash
pnpm build
pnpm start
```
